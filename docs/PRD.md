# PRD – 채용 브랜딩 기반 채용 사이트

## 0. 회사에 대한 간단한 소개 & MVP 포지셔닝
> **이 서비스는 ‘채용 브랜딩 콘텐츠와 채용 공고가 분절되어 후보자가 헤매는 문제’를 해결하기 위해 만들어진 채용 특화 커리어 사이트입니다.**  
> 회사 소개는 상단에서 짧고 명확하게 제공하고, 이후 직무 기반 탐색을 통해 후보자가 빠르게 지원 여부를 판단하도록 돕는 것을 MVP 목표로 합니다.

---

## 1. 프로젝트 한 줄 정의
> **후보자가 회사 정체성을 빠르게 이해한 뒤, 직무별로 채용 공고와 브랜딩 콘텐츠를 연결해 탐색하고 지원까지 이어지게 만드는 채용 브랜딩 커리어 사이트**

---

## 2. 타겟 사용자 페르소나

### 페르소나 A: 채용 담당자 (Primary User)
- **직무/역할**: Recruiter, People Ops, HR Manager
- **소속**: 스타트업 ~ 중견기업
- **목표**
  - 지원 전환률 개선
  - 채용 브랜딩 콘텐츠의 실질적 활용
- **페인 포인트**
  - 채용 공고와 콘텐츠가 분리되어 관리됨
  - 후보자가 채용 사이트에서 이탈하는 원인을 파악하기 어려움
  - 개발 리소스 없이도 직관적인 채용 사이트 운영 필요

### 페르소나 B: 지원 고려 중인 후보자 (End User)
- **직무/배경**: 개발, 마케팅, 제품 디자인 등
- **상태**: 적극 지원 전 / 탐색 단계
- **목표**
  - 회사가 어떤 곳인지 빠르게 이해
  - 내 직무와 관련된 정보만 선별적으로 확인
- **페인 포인트**
  - 회사 소개가 너무 길거나 추상적임
  - 공고와 콘텐츠를 오가며 길을 잃음
  - 불필요한 스크롤과 복잡한 페이지 구조

---

## 3. 핵심 기능 (우선순위 순)

### 1) 상단 회사 소개 + 직무 진입 CTA (Core)
- 회사 한 줄 소개(문제·미션 중심)
- “직무별 채용 보기” 명확한 CTA
- MVP 핵심 메시지 노출 영역

### 2) 직무 기반 탐색 허브
- 직무 선택 시 해당 직무의
  - 채용 공고
  - Crew Story(직무 인터뷰)
  - Inside Us(문화/영상)
- 콘텐츠 ↔ 공고 간 즉시 이동

### 3) 통합 검색 및 필터
- 채용 공고 + 콘텐츠 통합 검색
- 직무 / 키워드 / 경력 레벨 필터

### 4) 스크롤 최소화 UI 구조
- 탭 기반 정보 분리
- 요약 카드 및 핵심 정보 우선 노출

### 5) 콘텐츠 관리(CMS) – MVP 이후
- 인터뷰, 문화 콘텐츠 등록/수정
- 직무 태깅 기반 자동 노출

---

## 4. 화면 구성 (페이지 구조)

### 1) Home
- 회사 한 줄 소개 & 미션
- 직무별 채용 진입 CTA
- 주요 채용 공고 하이라이트
- 대표 Crew Story 카드

### 2) 직무 허브 페이지
- 직무 설명 요약
- 해당 직무 채용 공고 리스트
- 관련 Crew Story / Inside Us 콘텐츠

### 3) 채용 공고 상세 페이지
- 직무 역할 설명
- 자격 요건(Must / Nice)
- 채용 프로세스 안내
- 관련 콘텐츠 미리보기
- 지원 CTA

### 4) 콘텐츠 상세 페이지
- Crew Story / Inside Us 콘텐츠
- 핵심 요약
- 관련 채용 공고 CTA

### 5) 회사 소개 페이지 (About)
- 미션 / 비전 / 문화
- Inside Us 콘텐츠 모음
- 최근 뉴스/미디어 링크

---

## 5. 기술 스택 추천

### Frontend
- **Next.js (React)**: SEO 친화적, 페이지 단위 구조
- **Tailwind CSS**: 빠른 UI 구현, 트렌디한 디자인

### Backend
- **Node.js (NestJS 또는 Express)**
- **REST API 또는 GraphQL**

### Database
- **PostgreSQL**: 공고/직무/콘텐츠 관계 관리
- (옵션) **Supabase**: 빠른 MVP 구축(인증·스토리지 포함)

### 기타
- **Vercel**: 프론트엔드 배포
- **Headless CMS(Contentful, Sanity, Notion)**: 초기 콘텐츠 관리
