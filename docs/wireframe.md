# wireframe.md
채용 브랜딩 기반 채용 사이트 – 와이어프레임 초안 (MVP)

> 목적: 디자인이 아닌 **정보 구조(IA)·탐색 흐름·CTA 검증용 문서**  
> 원칙: 스크롤 최소화 / 직무 기준 탐색 / 공고↔콘텐츠 1클릭 이동

---

## 공통 UI 규칙 (Global)
- 상단 고정 Header
  - Logo
  - Jobs
  - Crew Story
  - Inside Us
  - About
  - Global Search
- 모든 상세 페이지에 **관련 공고 또는 관련 콘텐츠 CTA 필수**
- 긴 텍스트는 요약(3줄) + 더보기(accordion)
- 한 페이지에 긴 섹션 5개 초과 금지

---

## 1. Home

### 목적
- 회사 정체성을 10초 안에 이해
- 직무 선택으로 빠른 진입 유도

### 섹션 구성 (Top → Bottom)

#### 1) Hero (상단 대표 영역)
- 회사 한 줄 소개 (문제/미션 중심)
- 서브 문구: 어떤 사람들과, 어떤 방식으로 일하는지
- Primary CTA: [직무별 채용 보기]
- Secondary CTA: [회사 더 알아보기]

#### 2) Job Category Selector
- 버튼/카드: 개발 | 마케팅 | 디자인 | 기획 | 운영
- 클릭 시 → 해당 직무 허브 페이지 이동

#### 3) Featured Jobs
- 카드형 공고 3~5개
- 노출 정보: 직무 / 팀 / 경력 / 위치
- CTA: [공고 보기]

#### 4) Featured Crew Story
- 직무 인터뷰 카드 2~3개
- CTA: [이 직무 이야기 더 보기]

---

## 2. 직무 허브 페이지 (Job Hub)

### 목적
- 특정 직무 기준으로 공고 + 콘텐츠를 한 흐름으로 탐색

### 상단 영역
- 직무명 + 한 줄 설명
- 직무가 회사에서 해결하는 문제 요약

### 탭 구조 (스크롤 최소화)
| 채용 공고 | Crew Story | Inside Us |

#### 탭 1) 채용 공고
- 해당 직무 공고 카드 리스트
- CTA: [지원하기]

#### 탭 2) Crew Story
- 해당 직무 인터뷰 카드
- CTA: [인터뷰 읽기]
- 보조 CTA: [이 직무 채용 공고 보기]

#### 탭 3) Inside Us
- 문화/영상/블로그 카드
- 직무와 연관된 콘텐츠만 노출

---

## 3. 채용 공고 상세 페이지 (Job Detail)

### 목적
- 지원 결정을 빠르게 완료

### 고정 영역
- Job Summary Sticky Box
  - 직무명 / 팀 / 경력 / 위치
  - Primary CTA: [지원하기]

### 탭 구조
| 역할 | 자격 요건 | 채용 프로세스 | 관련 콘텐츠 |

#### 역할
- 이 포지션이 해결하는 문제
- 팀 내 역할

#### 자격 요건
- Must / Nice to have 구분

#### 채용 프로세스
- 단계 시각화: 서류 → 인터뷰 → 오퍼

#### 관련 콘텐츠
- Crew Story 카드
- Inside Us 카드
- CTA: [이 팀 더 알아보기]

---

## 4. 콘텐츠 상세 페이지 (Crew Story / Inside Us)

### 목적
- 사람·문화 이해 → 다시 채용 공고로 연결

### 섹션 구성

#### 1) Content Header
- 콘텐츠 제목
- 직무 / 팀 태그

#### 2) Main Content
- 인터뷰 텍스트 또는 영상

#### 3) Key Takeaway Box
- 이 팀의 일하는 방식 요약 (3줄)

#### 4) Related Jobs
- 관련 채용 공고 카드
- CTA: [이 팀 채용 공고 보기]

---

## 5. 회사 소개 페이지 (About)

### 목적
- 깊이 탐색용 회사 정보 제공

### 섹션 구성
- 회사 미션 / 비전
- Inside Us 콘텐츠 모음
- 최근 뉴스 / 미디어 링크

---

## 필수 사용자 흐름 체크 (Acceptance Criteria)
- Home → 직무 허브 → 공고 상세 → 지원 (끊김 없음)
- 콘텐츠 상세 → 관련 공고 1클릭 이동
- 직무 허브에서 스크롤 없이 공고/콘텐츠 전환 가능
